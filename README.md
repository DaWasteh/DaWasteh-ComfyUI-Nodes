DaWasteh’s ComfyUI Workflows

In diesem Repository veröffentliche ich meine selbst erstellten und angepassten ComfyUI-Workflows für verschiedene Bereiche der KI-gestützten Mediengenerierung.

Die Sammlung wird unregelmäßig erweitert und überarbeitet. Hinweise zu benötigten Modellen, Custom Nodes und besonderen Einstellungen befinden sich – soweit erforderlich – direkt bei den jeweiligen Workflow-Dateien.

Die Workflows dürfen gerne als Ausgangspunkt für eigene Projekte verwendet und angepasst werden. Da sich ComfyUI, Custom Nodes und Modelle laufend weiterentwickeln, kann eine dauerhafte Kompatibilität nicht garantiert werden.
